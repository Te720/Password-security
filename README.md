# Password-security

This is a script, which secure files by password GNU.

# Install

$ git clone https://github.com/Te720/Password-security.git

# Set excute permission

cd Password-security

chmod +x src/pass

# Move directories

mv etc/.raulm /etc

mv var/.tracer /var

mv src/pass /usr/bin
